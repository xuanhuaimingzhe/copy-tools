# ✏️ 文案工具箱 CopyTools

免费在线工具集，纯前端，零依赖（除二维码）。

## 工具列表

| 图标 | 工具 | 说明 |
|------|------|------|
| 📱 | 二维码生成 | 输入文字/网址，生成可下载的二维码 |
| { } | JSON格式化 | 格式美化、压缩、校验 |
| 🔐 | Base64编解码 | 文字与Base64互转，支持中文 |
| ⏰ | 时间戳转换 | Unix时间戳与日期时间互转 |
| 🎨 | 颜色转换 | HEX ↔ RGB ↔ HSL 格式互转 |
| 🔗 | URL编解码 | URL encode / decode |
| 🔍 | 正则测试 | 可视化正则匹配，支持多捕获组 |
| 🔑 | 密码生成 | 随机强密码生成，可自定义长度与字符集 |
| 📊 | 字数统计 | 中英文字数、字符数、段落数实时统计 |
| 📝 | Markdown编辑器 | 在线编辑、实时预览 |
