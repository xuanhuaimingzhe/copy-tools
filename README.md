# ✏️ 文案工具箱 CopyTools

免费在线文字处理工具集，纯前端，零依赖。

## 工具
- 📊 **字数统计** — 中英文字数、字符数、段落数实时统计
- 🚫 **敏感词检测** — 检测文本中的敏感词汇
- 🔄 **简繁转换** — 中文简体与繁体字一键互转
- { } **JSON 格式化** — 格式化、压缩、校验
- 🔐 **Base64 编解码** — 编码与解码，支持中文
- 📝 **Markdown 编辑器** — 在线编辑与实时预览

## 部署
推荐 Cloudflare Pages（国内可访问）：
1. 推送到 GitHub 仓库
2. Cloudflare Dashboard → Pages → 连接 Git 仓库
3. 框架选 None，输出目录留空 → 部署
